#### 基本使用

``` js
四种类型 error, warning, alarm, info
<div>
  <Label type="error">error</Label>
  <Label type="warning">warning</Label>
  <Label type="alarm">alarm</Label>
  <Label type="info">info</Label>
</div>
```
